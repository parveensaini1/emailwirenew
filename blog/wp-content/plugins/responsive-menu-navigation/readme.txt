== Description ==
**Description:**    

Responsive header top menu & navigation sidebar plugin for Wordpress. It is a perfect website header menu. More than 50 Customizeable settings. Flawless design. It is beautiful and works everywhere.You can choose every single color, so you can build your own custom design. You can also choose yourself what screen sizes it should be shown on.


**See it live here: [http://ahogenhaven.com/stuff/responsive-menu/](http://ahogenhaven.com/stuff/responsive-menu/)**


**Features:**
<ul>
<li>50+ Customizeable settings ($10 version feature)</li>
<li>You can literally choose colors on everything! ($10 version feature)</li>
<li>Widget support ($10 version feature)</li>
<li>Custom background image ($10 version feature)</li>
<li>Center title & tagline ($10 version feature)</li>
<li>Customer support ($10 version feature)</li>
<li>Lightweight, does not slow down pagespeed</li>
<li>You can add a logo or upload an image</li>
<li>Multi level menu hierarchy</li>
<li>Secondary menu which can be disabled</li>
<li>Addable custom Title & Description</li>
<li>Flawless animated design</li>
<li>Integrated search function which can be disabled</li>
<li>Addable background image or pattern with a custom height</li>
<li>Customize overlay colors and opacity   </li>
<li>The menu button can either always follow the screen when you scroll or stay the same place</li>
<li>Retina support</li>
<li>Customize what sizes the menu should be shown on</li>
</ul>


**Activation:** 
<ul>
<li>Upload & Activate the plugin</li>
<li>Go to appearance > Menu & choose a menu for responsive menu</li>
<li>Go to Settings > Responsive Menu & make your settings</li>
<li>Go to customize > Responsive Menu & pick your colors ($10 version feature)</li>
<li>Go to appearance > widgets and pick your widgets ($10 version feature)</li>
</ul>
